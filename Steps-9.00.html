On the Control Flow pane,  drag and drop one Execute SQL Task and a Sequence Container.

Drag and drop two Data Flow Tasks into the Sequence Container and name them as shown below




Double-click on the Execute SQL Task to Configure

Click on Connection to select the correction connection manager

Click on the 3 Dots in the SQL Statement box to write SQL Truncate statement for both staging tables

Click OK


Double-click on the DFT_LOE_Stg in the Sequence Container to move to the Data Flow Pane

Drag and drop the Excel Source, rename accordingly, and double-click to configure

Select the required connection manager

Select the Excel sheet

Click on the Columns option and make sure to select the required columns and click OK




Drag and Drop the Derived Column Transformation and double-click to update fields as shown below

Click OK


Drag and Drop the Conditional Split transformation, rename it, double-click, and configure

Use the IsNull function to separate Null values from good data

Rename fields as shown below


Drag and drop the Data Conversion and Union All transformations

Connect accordingly to separate the good data from bad data

Double-click on Data Conversion transformation and update data type as required and click OK


Drag and drop OLE DB Destination, connect and rename accordingly 

Double-click to configure

Select the right connection manager and click on the drop-down arrow to select the destination table

Click on the Map option to map columns  and click on OK


Run Select Statement in SSMS to verify proper load
