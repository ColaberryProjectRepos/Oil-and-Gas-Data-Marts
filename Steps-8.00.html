Create Connection Managers 8

Create project-level connection managers to enable source file access and destination load tables on the database in SQL Server Management Studio (SSMS).

  1. Creating Excel Connection Manager

Right-click on Connection Managers in the Solution Explorer and select New Connection Manager 


Select the Excel Connection Manager Type and click on ADD


Browse to select the Excel File(s) location, choose the Excel Version and click OK

Name the connection manager accordingly


       2.  Creating OLE DB Connection Manager for SQL Server

Right-click on Connection Managers in the Solution Explorer and select New Connection Manager 


Select the OLEDB Connection Manager Type and click on ADD






Choose the correct server name

Click on the drop-down arrow to select the database name where tables are stored

Click on Test Connection to make sure connection is established

Click OK




NOTE: Follow the same process to create a second Excel Connection Manager for Excel_Lease_OpEx and two other connection managers for SQLServe_Oil_Gas_ODS and SQLServe_Oil_Gas_DW
