Merge Transformation

  • Drag and drop the Merge transformation, connect, rename, and double-click to configure

          Double-click and check to make sure the work keys are mapped correctly then click OK

Sort transformation

  • Drag and drop the Sort transformation, connect, rename and double-click to configure

         Check the box in front of the Remove rows with duplicate sort values then click OK



Data Conversion

  • Drag and drop the Data Conversion, connect, double-click to configure

          Select required columns and change data type and length as needed then click OK

Derived Column

  • Drag and drop the Derived Column and double-click to configure

          Add new fields as required, add functions as required, and click OK
