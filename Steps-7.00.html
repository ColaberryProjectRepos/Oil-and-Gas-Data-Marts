Setting the pace: Create a New Project & New SSIS Package 7

STEP 1:

Open the Microsoft Visual Studio 

Click on New Project to create a new project




Select Integration Services

Name the project accordingly and Browse to select the appropriate location to save project and click on OK



To create a New SSIS package, right-click on SSIS Package and select New SSIS Package and then right-click on the new package and select Rename to rename package
